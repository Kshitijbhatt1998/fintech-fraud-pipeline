What's Inside

📊

Live Dashboard

Interactive Streamlit dashboard with daily fraud trends, risk breakdowns by product, card type, and email domain.

🤖

XGBoost Model

58-feature gradient boosting classifier with time-series cross-validation, achieving 97.91% holdout AUC on IEEE-CIS data.

🔧

dbt Pipeline

Fully tested staging → mart SQL transformations with 12 built-in data quality tests and incremental materialization.

📊

MLflow Tracking

Experiment tracking logs parameters, fold-level AUC scores, and model artifacts. Full auditability of every training run.

⚡

DuckDB Backend

In-process OLAP engine. No server required — query 590K transactions in milliseconds from a single portable file.

✍

CI/CD Tested

GitHub Actions runs dbt tests and Python/SQL linting on every PR. No broken pipelines reach main.
